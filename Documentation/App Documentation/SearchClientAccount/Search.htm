Search Client Account

To search for a specific client or specific demographic groups, you can use the following categories in your search:

  • ID
  • Name
  • First name
  • Birthdate
  • E-Mail
  • Place
  • ZipCode
  • Gender

If you don't select a categorie, the program will search your term in every category!
